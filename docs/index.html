How to install and use

  1. Click Download installer for the setup wizard, or Download Weather Wizard.exe launcher for the zipped launcher folder.
  2. Open the downloaded setup file. If Windows SmartScreen appears, choose “More info” then “Run anyway” because this build is unsigned.
  3. Follow the setup wizard and keep “Create desktop shortcut” enabled.
  4. Open Weather Wizard.exe.
  5. Choose Guest mode, Google, Discord, or email on the home page.
  6. Click Use my location to get weather where you live, or search any city.
  7. Click Allow notifications if you want weather notification popups.
